# To-Do List App

A simple command line To-Do list application in Python that saves tasks to a file.

## Features
- Add tasks
- Delete tasks
- Persistent tasks (saved in `tasks.txt`)
- List all tasks

## Run
```bash
python todo.py

# GUI To-Do List App

A Python GUI To-Do List app using Tkinter. Users can choose where to save tasks.

## Features
- Add tasks
- Delete tasks
- Save/load tasks from any file
- Persistent data

## Run
```bash
python gui_todo.py
